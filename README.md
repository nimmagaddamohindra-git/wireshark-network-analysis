# Network Traffic Analysis using Wireshark

## 📌 Project Overview
This project demonstrates basic network traffic analysis using Wireshark. The goal was to capture and analyze real-time network packets and understand how systems communicate over the internet.

## 🛠 Tools Used
- Wireshark
- Windows OS

## 🔍 Key Activities

### 1. Packet Capture
- Captured live network traffic using Wireshark
- Observed real-time packet flow

### 2. IP Filtering
- Applied filters to isolate system-specific traffic:
- - Analyzed:
  - Source IP (sender)
  - Destination IP (receiver)

### 3. DNS Analysis
- Applied DNS filter:
- - Observed domain resolution for:
  - swiggy.com
  - mcdonalds.com
  - microsoft services

### 4. Traffic Investigation
- Used search filters:
- - Identified real-world application traffic

## 🧠 Key Learnings
- Understanding of network communication (source & destination)
- DNS resolution process (domain → IP)
- Packet-level analysis using Wireshark
- Filtering techniques for isolating relevant traffic
